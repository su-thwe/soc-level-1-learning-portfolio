# SOC Level 1 Learning Portfolio

This repository documents my SOC Level 1 learning through security notes, practical workflows, incident reports, and small portfolio projects.

The goal of this portfolio is to show how I am building foundational blue-team skills in areas such as incident response, phishing triage, network traffic analysis, threat detection, and security documentation.

## Focus Areas

- SOC analyst fundamentals
- Incident documentation
- Phishing and social engineering analysis
- Network traffic analysis
- Security investigation workflows
- Blue-team reporting and remediation

## Repository Structure

| Folder | Purpose |
|---|---|
| `rooms/` | Learning notes and writeups from SOC-related rooms/modules |
| `projects/` | Practical security workflows, reports, and portfolio outputs |
| `templates/` | Reusable templates for reports, room writeups, and investigations |
